# Changelog

All notable public-facing changes to SelfHarness are tracked here. The project
is pre-1.0, so compatibility guarantees are limited to the stable API described
in [README.md](README.md).

## Unreleased

- Added public repository documentation index, governance notes, contribution
  guide, security policy, code of conduct, and visual architecture assets.
- Made MIT licensing explicit through the root license file, package metadata,
  README, and docs index.
- Documented the code CLI control plane with provider/model/effort selection,
  thread controls, slash commands, and interrupt behavior.

## 0.1.0

- Initial public package metadata and CLI entrypoint.
- Paper-faithful deterministic Self-Harness loop with repeat-aware evaluation,
  proposer context, split validation, and audit artifacts.
- Trusted-corpus, local verifier, release-smoke, readiness, and reproduction
  boundary tooling.
